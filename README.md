# ✨ Magical Login Lamp

A beautiful and animated login page with a glowing magical lamp effect, interactive UI, glassmorphism design, and starry animated background built using HTML, CSS, and JavaScript.

---

## 🌟 Features

- 💡 Interactive glowing lamp
- ✨ Magical login animation
- 🌌 Animated star background
- 🎨 Glassmorphism UI design
- 📱 Responsive layout
- ⚡ Pure HTML, CSS & JavaScript
